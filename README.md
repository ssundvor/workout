# Vanity Sprint

A phone-first, offline workout tracker for Scott's fixed 8-week Vanity Muscle Sprint.

## What it does

- Tracks Monday, Tuesday, Thursday, and bonus workouts
- Prefills each set from the last session
- Handles supersets in alternating order
- Starts the correct rest timer after every logged set
- Saves the active workout and history to local storage on every tap
- Works offline after the first visit

The app is a zero-dependency static site. Open `index.html` locally or publish the repository with GitHub Pages.
